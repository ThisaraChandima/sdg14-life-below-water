# Contributing to SDG14: Life Below Water

Welcome to the team! Since there are 5 of us working on this project, we are using a **Branch and Pull Request Workflow** to keep our code organized and prevent conflicts.

Please do **NOT** push your changes directly to the `main` branch. 

## How to Contribute (VS Code Instructions)

Follow these steps every time you want to work on a new feature or fix a bug:

### 1. Get the latest code
Before starting any new work, make sure your local repository is up to date with the main project.
* Open the VS Code terminal.
* Run: `git pull origin main`

### 2. Create a new branch
Create a separate branch for your specific task. Name it something descriptive.
* In the VS Code terminal, run: `git checkout -b yourname/feature-name`
  *(Example: `git checkout -b john/add-footer`)*

### 3. Make your changes
* Edit the HTML/CSS files in VS Code.
* Save your work.

### 4. Commit and Push
* Go to the **Source Control** tab in VS Code (the branch icon on the left).
* Stage your changes (click the `+` next to the files).
* Type a short, clear commit message (e.g., "Added footer to index.html").
* Click **Commit**.
* Click **Publish Branch** (or in the terminal, run `git push origin yourname/feature-name`).

### 5. Open a Pull Request
* Go to our repository on GitHub: https://github.com/ThisaraChandima/sdg14-life-below-water
* You will see a green button that says **Compare & pull request** for your newly pushed branch. Click it.
* Add a title and description of what you changed.
* Click **Create pull request**.

### 6. Review and Merge
* **Thisara** will review your Pull Request.
* Once approved, Thisara will merge it into the `main` branch!
