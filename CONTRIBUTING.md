# Contributing to SDG14: Life Below Water

Welcome to the team! Since there are 5 of us working on this project, we are using a **Branch and Pull Request Workflow** to keep our code organized and prevent conflicts.

Please do **NOT** push your changes directly to the `main` branch. 

## How to Contribute (VS Code Instructions)

Follow these steps every time you want to work on a new feature or fix a bug. You can do almost all of this directly inside the VS Code app!

### Step 1: Get the latest code
Before starting, make sure you have the latest changes.
1. Open the project in VS Code.
2. In the bottom-left corner of VS Code, make sure it says `main`. 
3. Click the **Sync Changes** button (or click `...` in the Source Control panel and select `Pull`) to download any new updates.

### Step 2: Create a separate branch (The most important step!)
Because the `main` branch is locked, you *must* create your own branch to work on.
1. Click on the word `main` in the bottom-left corner of VS Code.
2. A menu will pop up at the top. Click **+ Create new branch**.
3. Type a name for what you are working on (for example: `sarah/homepage-css` or `fix-image-bug`) and hit Enter.

### Step 3: Write code and Commit
1. Make your changes to the HTML/CSS files just like normal.
2. When you are ready to save, go to the **Source Control** icon on the left toolbar (it looks like a little branch/fork).
3. Stage your changes by hovering over your changed files and clicking the `+` button.
4. Type a message explaining what you did in the text box (e.g., "Added styling to the team page").
5. Click the blue **Commit** button.

### Step 4: "Push" and ask to Merge
1. After committing, there will be a blue button that says **Publish Branch**. Click that. This sends your branch to GitHub.
2. Now, go to our repository on the GitHub website: https://github.com/ThisaraChandima/sdg14-life-below-water
3. GitHub will show a big green button that says **"Compare & pull request"**. Click it!
4. Leave a little note explaining what you changed, and click **Create pull request**.

### Step 5: Review and Merge
* **Thisara** will get a notification, review your code, and click the **Merge** button to add it to the main project!
